# CEDA

**Center for Educational Data Analytics** — van data naar betrouwbare inzichten voor het onderwijs.

CEDA ontwikkelt samen met mbo, hbo en wo tools en analyses waarmee onderwijsinstellingen grip houden op de toekomst. Alle code is open source en te vinden op [github.com/cedanl](https://github.com/cedanl).

---
