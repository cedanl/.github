    CEDA

    Center for Educational Data Analytics — van data naar betrouwbare inzichten voor het onderwijs.

    CEDA ontwikkelt samen met mbo, hbo en wo tools en analyses waarmee onderwijsinstellingen grip houden op de toekomst. Alle code is open source en te vinden op github.com/cedanl.

    Analytics

    • Instroomprognose

      Voorspellen van de instroom van studenten op basis van Studielink-telbestanden.

      GitHub

    • Wisselstroom

      Studentstromen door het onderwijs analyseren met DUO-bekostigingsbestanden.

      Package · Demo

    • Onderwijsdata Chat

      Chat met open Nederlandse onderwijsdata (CBS + RIO).

      GitHub

    • Uitnodigingsregel

      Voorspellen van studentsuitval in het eerste jaar.

      GitHub

    • No Fairness Without Awareness

      MVP voor fairness-analyses op onderwijsdata.

      GitHub
